# Job Finder AI

An AI-powered job discovery and application assistant.

## What’s inside

- **Backend**: scraping, matching/ranking, job persistence, scheduling
- **Agents**: modular AI agents to process job data
- **Frontend**: UI for browsing jobs and application workflow
- **Notifications**: alerts for new matches and follow-ups

## Quick start

1. Copy env:
   ```bash
   copy .env.example .env
   ```
2. See docs:
   - `docs/00_PROJECT_OVERVIEW.md`
   - `docs/05_ARCHITECTURE.md`
   - `docs/08_API.md`

## Development

Run your backend/frontend as described in the relevant documentation files.
